### 3.2 各因子详细说明

#### 3.2.1 资金流因子 (20%)

```python
class MoneyFlowAnalyzer:
    """资金流向分析"""
    
    分析内容:
    - 超大单净流入
    - 大单净流入
    - 主力持仓占比
    - 散户比例
    
    评分逻辑:
    - 主力净流入>5000万 → +30分
    - 主力持仓>50% → +15分
    - 散户净流入>5000万 → -20分
    
    输出:
    - money_flow_score: 0-100分
    - is_good_flow: bool
    - is_bad_flow: bool
```

#### 3.2.2 动量因子 (20%)

```python
class MomentumFactor:
    """动量因子 - 追涨杀跌顺势而为"""
    
    包含指标:
    1. RPS (Relative Price Strength)
       - 相对价格强度
       - 20日涨幅排名百分比
    
    2. HMMI (Horizontal Momentum Index)
       - 水平动量指数
       - 衡量趋势延续性
       - 正收益占比 > 70% → 强势上涨
    
    3. 多周期收益率
       - 5日/10日/20日收益率
    
    评分逻辑:
    - RPS > 85 → 95分
    - HMMI > 70 → 趋势强劲
```

#### 3.2.3 价值因子 (15%)

```python
class ValueFactor:
    """价值因子 - PE/PB/ROE筛选"""
    
    包含指标:
    1. PE评分 (市盈率)
       - PE < 10 → 90分
       - PE < 20 → 75分
       - PE > 50 → 40分
    
    2. PB评分 (市净率)
       - PB < 1 → 95分
       - PB < 2 → 80分
       - PB > 5 → 45分
    
    3. ROE评分 (净资产收益率)
       - ROE > 20% → 95分
       - ROE > 15% → 85分
       - ROE < 5% → 40分
```

#### 3.2.4 北向资金 (10%)

class NorthboundFlow:

class HolidayEffectModel:
    """节假日效应模型"""
    
    效应类型:
    
    1. 春节效应 (1-2月)
       - 节前减仓持币

       - 机构调仓
       - 波动加大
    
    5. 月末效应 (25日后)
       - 资金面紧张
    
    仓位调整:
    - HIGH风险 → 50%仓位
    - MEDIUM风险 → 80%仓位
    - LOW风险 → 100%仓位

### 4.3 黑天鹅机制

```python
class BlackSwanProtection:
    """黑天鹅机制 - 极端行情保护"""
    
    检测项:
    
    1. 连续涨跌停
       - 连续2日跌停 → +40分风险
       - 连续3日涨停 → +20分风险(追高)
    
    2. 波动率飙升
       - 当前波动率/历史波动率 > 2.5 → +25分风险
       - > 3.0 → EXTREME级别
    
    3. 成交量萎缩
       - 量比 < 0.3 → +15分风险
    
    风险等级:
    - 0-20分 → 🟢低风险 → 正常交易
    - 20-40分 → 🟡中风险 → 谨慎观望
    - 40-60分 → 🟠高风险 → 减仓止损
    - 60+分 → 🔴极高风险 → 立即止损回避
```

## 五、止损与仓位体系

### 5.1 ATR动态止损

```python
def calculate_dynamic_stop_loss(price, df):
    """ATR动态止损"""
    
    # ATR = Average True Range (14日)
    # 止损位 = 当前价 - (ATR × 2)
    
    假设:
    - 浪潮信息当前价: 67.50元
    - ATR(14日): 约5.21元
    - 止损位: 67.50 - 5.21×2 = 62.29元
    - 止损幅度: 7.7%
    
    目标位:
    - 第一目标: 止损位 + ATR×2
    - 第二目标: 止损位 + ATR×4
```

### 5.2 凯利公式仓位

```python
def calculate_kelly_position(price, stop_loss, win_rate, profit_ratio, max_position):
    """
    凯利公式仓位管理
    f = (p×b - q) / b
    
    其中:
    - p = 胜率 (55%)
    - q = 败率 (45%)
    - b = 盈亏比 (2.0)
    
    凯利公式: f = (0.55×2.0 - 0.45) / 2.0 = 32.5%
    安全版本: 50%凯利 = 16.25%
    上限: 20%
    """
    
    输出:
    - recommended_position: 建议仓位百分比
    - kelly_fraction: 凯利分数
```

### 5.3 止损冷却期

class StopLossCooldown:

    - 冷却期结束后恢复正常
