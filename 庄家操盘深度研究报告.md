## 七、庄家操盘规律总结

### 1. 航天电器操盘特征
```
吸筹阶段: 缩量横盘 -> 突然放量拉升
拉升阶段: 连续放量上涨 -> 涨停
派发阶段: 高位震荡 -> 放量出货
洗盘特征: -2%~-5%回调是买入机会
```

### 2. 最佳买入时机
- 放量上涨突破日买入
- 回调至MA5支撑买入
- 涨停后次日有惯性

### 3. 止损纪律
- 单只: 亏损8%止损
- 大盘: 跌破MA20离场
- 情绪: 跌停不要抄底
