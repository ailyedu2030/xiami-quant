## 一、系统概述

### 1.1 核心定位

虾米股票智能分析系统是一个**专业级A股智能决策系统**，通过多智能体协同工作，模拟真实投资决策团队的工作流程，为投资者提供科学、客观的股票分析建议。

### 1.2 设计理念

```
┌─────────────────────────────────────────────────────────────┐
│                    模拟专业投资团队                          │
├─────────────────────────────────────────────────────────────┤
│  12个专项分析师 → 4个专业委员会 → 首席决策官                │
│                                                              │
│  每个人只做自己擅长的事，整体大于部分之和                     │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 核心特点

| 特点 | 说明 |
|------|------|
| **多智能体协同** | 16个智能体分工合作，各司其职 |
| **实时数据** | 每次分析先获取实时行情 |
| **战法融合** | 缠论、均线、形态、短线等经典战法 |
| **风险控制** | 独立风控委员会，止损位明确 |
| **决策可追溯** | 每一步决策都有明确依据 |

## 二、系统架构

### 2.1 三层决策架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Layer 1: 专项智能体层                     │
│                       (12个智能体)                          │
├─────────────────────────────────────────────────────────────┤
│  趋势委员会(3)     │ 价值委员会(3)   │ 风控委员会(3) │ 时机委员会(3) │
│  ─────────────     │ ─────────────   │ ───────────── │ ───────────── │
│  • 缠论专家        │ • 基本面专家    │ • 支撑阻力专家│ • 短线专家    │
│  • 均线专家        │ • 技术专家      │ • 回测专家    │ • 尾盘专家    │
│  • 形态专家        │ • 筹码专家      │ • 国际专家    │ • 情绪专家    │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    Layer 2: 专项委员会层                     │
│                      (4个委员会)                             │
├─────────────────────────────────────────────────────────────┤
│  趋势委员会 ← 判断方向      │ 价值委员会 ← 判断估值          │
│  风控委员会 ← 判断风险      │ 时机委员会 ← 判断时机          │
│                                                              │
│  各委员会独立投票，权重不同                                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                   Layer 3: 裁决与决策层                      │
├─────────────────────────────────────────────────────────────┤
│  裁决委员会: 汇总四委员会意见，按权重投票                     │
│  首席决策官: 最终拍板，拥有一票否决权                         │
└─────────────────────────────────────────────────────────────┘
                              ↓
                         最终决策
                   买入 / 观望 / 回避
```

## 三、核心战法

### 3.1 缠论三类买点

```
┌─────────────────────────────────────────────────────────────┐
│  第一类买点: 趋势背驰点，底部确认                          │
│  ├── 价格创新低                                              │
│  ├── MACD黄白线未同步创新低（底背离）                        │
│  └── 评分权重: 40%                                          │
├─────────────────────────────────────────────────────────────┤
│  第二类买点: 反弹后回调确认点，最安全                        │
│  ├── 不破第一类买点                                         │
│  ├── 趋势确认信号                                           │
│  └── 评分权重: 85%（最推荐）                                │
├─────────────────────────────────────────────────────────────┤
│  第三类买点: 突破中枢后的回踩点                             │
│  ├── 价格突破近期高点                                        │
│  ├── 回踩不破中枢上沿                                       │
│  └── 评分权重: 75%                                          │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 2560战法

```
核心指标:
  • MA25 (25日均线): 判断趋势方向
  • VOL_MA5 (5日均量): 短期资金活跃度
  • VOL_MA60 (60日均量): 中长期资金动向

买入条件:
  ① MA25 必须向上
  ② VOL_MA5 上穿 VOL_MA60（金叉）

卖出条件:
  ① VOL_MA5 下穿 VOL_MA60（死叉）
  ② MA25 开始走平/向下
```

### 3.3 尾盘买入法

```
时间窗口: 14:30 - 15:00

选股条件:
  ① 涨幅3%-5%（最佳区间）
  ② 量比>1.5
  ③ 股价在MA5上方
  ④ K线收阳，上影线短

次日操作:
  开盘或早盘卖出
```

### 3.4 支撑阻力战法

```
支撑位计算:
  1. 前期高低点
  2. 均线系统 (MA5/10/20/60)
  3. 布林带上下轨
  4. 成交密集区

操作建议:
  • 回调到支撑位买入
  • 跌破支撑位止损
  • 突破阻力位加仓
```

## 四、决策流程

### 4.1 四委员会投票规则

| 委员会 | 权重 | 评分>=70 | 评分50-70 | 评分<50 |
|--------|------|----------|-----------|---------|
| 趋势委员会 | 30% | 🟢 上涨 | 🟡 震荡 | 🔴 下跌 |
| 价值委员会 | 25% | 🟢 低估 | 🟡 合理 | 🔴 高估 |
| 风控委员会 | 25% | 🟢 低风险 | 🟡 中风险 | 🔴 高风险 |
| 时机委员会 | 20% | 🟢 时机成熟 | 🟡 观望 | 🔴 等待 |

### 4.2 首席决策官否决条件

```
❌ 否决条件1: 综合评分 < 40分
❌ 否决条件2: 回避票占比 >= 50%
❌ 否决条件3: 现价高于建议买入点5%以上
```

### 4.3 最终决策判定

```
买入条件:
  ① 买入票占比 >= 60%
  ② 综合评分 >= 50分
  ③ 未触发否决条件

观望条件:
  ① 观望票占比 >= 50%
  ② 或买入/观望票合计不足60%

回避条件:
  ① 回避票占比 >= 50%
  ② 或触发任何否决条件
```
