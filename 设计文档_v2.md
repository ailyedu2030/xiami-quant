## 四、核心算法

### 4.1 ATR动态止损

```python
止损位 = 当前价格 - (ATR × 2)
目标1  = 当前价格 + (ATR × 2)
目标2  = 当前价格 + (ATR × 3)
目标3  = 当前价格 + (ATR × 5)
```

### 4.2 凯利公式

```python
f* = (bp - q) / b

其中:
  f* = 仓位比例
  b  = 赔率（盈利/亏损）
  p  = 胜率
  q  = 1 - p

安全仓位 = f* × 50%  # 只用一半
最大仓位 = 20%
```

### 4.3 否决条件

```
❌ 综合评分 < 40分 → 否决
❌ 回避票占比 >= 45% → 否决
❌ 现价高于建议买入点8% → 否决
❌ ATR > 股价8% → 否决（波动过大）
```
