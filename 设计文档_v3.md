## 七、核心算法

### 7.1 ATR动态止损

```python
止损位 = 买入价 - 2 × ATR
目标位 = 买入价 + 3 × ATR
移动止盈 = 盈利>10% 且回撤≥5%
```

### 7.2 凯利公式

```python
f* = (p × b - q) / b

其中:
  f* = 最优仓位比例
  b  = 赔率 (盈利/亏损)
  p  = 胜率
  q  = 1 - p

安全仓位 = f* × 50%
熊市仓位 = 安全仓位 × 70%
```

### 7.3 VaR计算

```python
VaR(95%) = 历史模拟法
CVaR(95%) = 条件在险价值
蒙特卡洛 = 10000次模拟
```

## 九、统计学指标

### 9.1 收益率分析

| 指标 | 计算方法 |
|:---|:---|
| 均值 | E[R] = ΣR/n |
| 方差 | Var(R) = Σ(R-E[R])²/(n-1) |
| 偏度 | Skew = E[(R-μ)³]/σ³ |
| 峰度 | Kurt = E[(R-μ)⁴]/σ⁴ - 3 |

### 9.2 风险指标

| 指标 | 公式 |
|:---|:---|
| VaR(95%) | Percentile(returns, 5%) |
| CVaR(95%) | Mean(returns[returns ≤ VaR]) |
| 夏普比率 | (Rp - Rf) / σp |
| 信息比率 | (Rp - Rb) / TrackingError |
